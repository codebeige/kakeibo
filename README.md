# kakeibo

Manage shared budgets online.

## Getting started

Run the project directly:

    boot run

Run the project's tests (they'll fail until you edit them):

    boot test

Build an uberjar from the project:

    boot build

Run the uberjar:

    java -jar target/kakeibo-0.1.0-SNAPSHOT-standalone.jar
